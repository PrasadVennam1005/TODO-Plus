### 🔥 What’s New in Version [VERSION]?

⚡ Live Real-Time Auto-Scan: The TODO tree updates live as you type in any editor window. Zero manual refreshes needed!
🖱️ Multi-Selection & Batch Actions: Select multiple TODO items or group headers to complete or revert in batch.
📋 1-Click "Copy for Standup": Instantly format selected tasks into clean Markdown/Slack checklists for daily standup updates.
📝 Multi-Line TODO Comments: Automatically parses indented bullet points and continuation lines beneath comments.
⚙️ Custom Keyword Recognition: Recognizes HACK:, BUG:, NOTE:, and OPTIMIZE: alongside standard TODO and FIXME.
🔊 Interactive Audio Feedback: Hear a satisfying audio chime when checking off completed tasks!
📄 Executive PDF & HTML Reports: Export printable task reports with visual metric cards and priority charts.
🌐 Expanded Solution & .CSHTML Support: Added "Entire Solution / Project" scope for Rider, ASP.NET Core (.cshtml/.razor), Vue, Svelte, Dart, and 15+ languages.
💡 Quick Fixes (Alt + Enter): Toggle task completion directly from your code editor.
